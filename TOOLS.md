# TOOLS.md - 本地笔记

“技能”（Skills）定义了工具的*运作方式*。本文件专用于记录*你的*具体细节——即那些专属于你个人配置的内容。

## 此处应包含的内容

例如：

- 摄像头的名称及位置
- SSH 主机及别名
- TTS（文本转语音）的首选语音
- 音箱/房间的名称
- 设备的昵称
- 任何特定于你所处环境的设置

## 示例

```markdown
### 摄像头

- living-room → 主要活动区，180° 广角
- front-door → 入口处，支持运动感应触发

### SSH

- home-server → 192.168.1.100，用户：admin

### TTS

- 首选语音：“Nova”（音色温暖，略带英式口音）
- 默认音箱：厨房的 HomePod

```

## 为何要分开存放？

“技能”是共享的，而你的配置则是你独有的。将两者分开存放，意味着你在更新技能时不会丢失自己的笔记，同时在分享技能时也不会无意中泄露你的基础设施细节。

---

凡是对你工作有帮助的内容，尽可添加于此。这便是专属于你的“备忘录”。
